Ce face

rotv-mcp conectează ChatGPT (și orice alt client MCP) la grila live a ~254 canale TV din România și la cataloagele de streaming disponibile aici: Netflix, HBO Max, Disney+, Prime Video, Apple TV+, SkyShowtime. 13 tool-uri pentru căutare, recomandare, planificare seară și — cel mai important — decizie unică contextuală.

Tool-ul principal tv_concierge răspunde la întrebarea care contează cu adevărat: „Am 2 ore din viață. Care e cea mai bună utilizare a lor din tot ce e disponibil acum?"

Cum lucrează concierge-ul

  • Multi-source — combină TV (EPG live) + streaming (catalog România) în același pool de scoring
  • Anti-noise filter — exclude default politică, reality, talk-show, știri (cu count vizibil în output)
  • Confidence engine pe 5 axe — rating signal + mood fit + time fit + availability + opportunity cost → procentaj 0–100 cu breakdown
  • Dedup pe titlu — ~46% din EPG sunt repetări; păstrăm winner-ul, restul devin dup_other_airings
  • Opportunity-cost lookahead — semnalează dacă ceva semnificativ mai bun începe la sfârșitul ferestrei
  • Diverse alternatives — până la 3 alternative cu pro/contra explicite și motivul pentru care NU au fost prima alegere
  • Timeline cu pauze — inserare automată de pauze între segmente (≥ 5 min)

Conectează serverul la asistentul tău AI

Server public, read-only, fără cheie și fără cont — orice client MCP compatibil se poate conecta. Rețetele de mai jos sunt testate pe acest server sau pe serverele-surori din aceeași familie.

Gemini CLI

  1. Instalează CLI-ul: npm install -g @google/gemini-cli
  2. Adaugă serverul în ~/.gemini/settings.json: 
    {
  "mcpServers": {
    "rotv": { "httpUrl": "https://tv.madeinro.eu/mcp" }
  }
}
  3. Pornește gemini și întreabă: „ce e important diseară la TV?" — Gemini alege singur tv_important_today / tv_concierge (testat pe 10 iul 2026, apeluri vizibile în log-urile serverului)
Aplicația Gemini (gemini.google.com/apps → custom app link) acceptă și ea MCP-uri custom, dar momentan doar în SUA, cu abonament Ultra — în EEA folosește Gemini CLI.

Claude Code / claude.ai

  1. În Claude Code, o singură comandă: claude mcp add --transport http rotv https://tv.madeinro.eu/mcp
  2. Pe claude.ai: Settings → Connectors → Add custom connector → URL-ul de mai sus, fără autentificare

ChatGPT

  1. Deschide ChatGPT și mergi la Settings → Connectors (sau Apps & Connectors, în funcție de versiune)
  2. Click pe Add custom connector (sau + New connector)
  3. În câmpul Server URL introdu: https://tv.madeinro.eu/mcp
  4. În Authorization: lasă pe None (server-ul e public read-only)
  5. Click Connect. La aprobare ar trebui să vezi 13 tool-uri disponibile, inclusiv tv_concierge
Dacă tool-urile par cache-uite (vezi mai puține sau lipsește tv_important_today): Disconnect + Reconnect cu aceeași URL — clientul face fresh tools/list la reconectare.

Diagnostice live

Endpoint-ul /mcp/health returnează JSON cu starea curentă a serverului:

curl https://tv.madeinro.eu/mcp/health
{
  "ok": true,
  "service": "rotv-mcp",
  "version": "3.0.x",
  "uptime_s": 1234,
  "cache_loaded_at": "2026-06-05T18:00:00.000Z"
}

Pentru a verifica freshness-ul datelor direct (fără ChatGPT):

curl -X POST https://tv.madeinro.eu/mcp \
  -H "Content-Type: application/json" \
  -H "Accept: application/json, text/event-stream" \
  -d '{"jsonrpc":"2.0","id":1,"method":"tools/call",
       "params":{"name":"tv_check_freshness","arguments":{}}}'

Arhitectură pe scurt

  • Node.js v24 + Express + Model Context Protocol SDK (StreamableHTTPServerTransport, stateless)
  • Read-only consumer al datelor scrise de alt serviciu pe același VPS (no DB, no pipeline duplication)
  • JSON file cache in-memory cu fs.watch debounced + safety-net 5min
  • Cloudflare tunnel path-based ingress: tv.madeinro.eu/mcp/* → localhost:3010
  • pm2 fork mode, zero-downtime reload, max_memory_restart 256M
  • Versioned outputSchemas declarate pentru toate 13 tool-uri (ChatGPT validează structurat răspunsurile)

De ce există

rotv-mcp începe cu observația simplă: utilizatorul tipic petrece 30 de minute alegând între TV, Netflix, HBO, Disney+, YouTube. Problema nu e lipsa de conținut. E excesul.

Concierge-ul nu e un ghid TV. Nu listează. Nu sugerează 10 opțiuni. Devine filtru personal de zgomot și răspunde la întrebarea practică: „Având tot ce e disponibil acum, ce alegi pentru următoarele X minute din viața mea?"

Și o spune cu confidence%, anti-noise count, dedup transparent și alternative care nu sunt doar „următoarele pe listă", ci aleasă pe buckets (sursă diferită / mai lung / mai scurt cu rating mai mare / gen diferit).
