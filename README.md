# rotv-mcp

Read-only Model Context Protocol (MCP) server exposing live Romanian TV schedule and streaming catalog to MCP clients (ChatGPT custom connectors, Claude Desktop, etc.).

**Production URL:** `https://tv.madeinro.eu/mcp`
**Transport:** Streamable HTTP (POST JSON-RPC, JSON or SSE response). Stateless.
**Auth:** Public (no auth) by default. Optional static bearer via `MCP_AUTH_TOKEN` env.

## Data sources

Reads three JSON files atomically generated by the existing `rotv-guide` app (NOT modified by this service):

- `/opt/apps/rotv-guide/public/data/epg-normalized.json` — 258 TV channels, −6h to +72h, ISO 8601 UTC times
- `/opt/apps/rotv-guide/public/data/epg-homepage.json` — 15 main channels, −2h to +36h
- `/opt/apps/rotv-guide/public/data/streaming-full.json` — Netflix/HBO Max/Prime/Disney+/Apple TV+ catalogs

Files are loaded on startup, refreshed on `fs.watch` events (debounced 750ms) with a 5-minute safety-net.

## Tools

| Tool | Purpose |
|---|---|
| `tv_now_on_tv` | What is broadcasting right now |
| `tv_search_program` | Search by title / channel / category / timeframe |
| `tv_get_prime_time` | 20:00–23:00 Europe/Bucharest lineup |
| `tv_recommend_today` | Deterministic recommendations (films/docs preferred, news excluded by default) |
| `tv_get_title_details` | Lookup a title across TV airings + streaming providers |

Time references accepted: `now`, `tonight`, `tomorrow`, `weekend`, `primetime`, or ISO 8601 instant/range.

## Operate

```bash
# start under pm2
pm2 start ecosystem.config.cjs && pm2 save

# tail logs
pm2 logs rotv-mcp --lines 50

# stop without forgetting
pm2 stop rotv-mcp

# fully remove
pm2 delete rotv-mcp && pm2 save

# health
curl https://tv.madeinro.eu/mcp/health
```

## Routing

Cloudflare tunnel (`/etc/cloudflared/config.yml`) routes `tv.madeinro.eu/mcp/*` → `localhost:3010` via a path-based ingress rule placed ABOVE the existing `tv.madeinro.eu` rule. The existing `rotv-guide` app on port 3002 is unaffected.

Rollback: `sudo cp /etc/cloudflared/config.yml.pre-mcp /etc/cloudflared/config.yml && sudo systemctl reload cloudflared`.
